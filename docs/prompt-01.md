## Prompt 01 — Tutorial Storage Update

### 목표
- 튜토리얼 진행/완료 상태를 버전과 함께 안전하게 저장하고, localStorage를 사용할 수 없는 환경에서도 동일한 흐름을 유지합니다.

### 진행 내용
1. `assets/js/tutorial/tutorialStorage.js`를 추가해 튜토리얼 상태 저장 유틸리티를 구현했습니다.
2. 스토리지 키와 현재 버전을 상수로 관리하고, 버전 불일치 시 상태를 초기화하는 `ensureTutorialVersion`을 제공했습니다.
3. 모든 읽기/쓰기에 `try/catch`를 적용해 localStorage 예외 시 메모리(Map) 스토어로 자동 전환합니다.
4. 진행 단계는 1~4로 클램프하고, 비정상 값이 들어와도 정상 범위로 교정하도록 했습니다.
5. 완료/진행/진행중 플래그, 버전, 초기화 API를 노출해 “최초 강제 튜토리얼”이 fallback 환경에서도 동작하도록 보장합니다.

### 테스트 메모
- 브라우저 없는 환경이므로 수동 테스트는 수행하지 않았습니다. 이후 통합 시 튜토리얼 흐름과 저장 동작을 브라우저에서 확인 예정입니다.
