# Tutorial Expansion – Step 7 & 8

## 작업 요약
- 게임 루프에 튜토리얼 분기와 성공 판정/재시도 흐름을 연결해 단계별 진행 조건을 반영했습니다.
- 튜토리얼 중 사망 시 일반 게임오버 흐름을 우회하고 즉시 재시도로 돌아가도록 처리했습니다.

## 세부 변경 내역
1. **Loop 튜토리얼 분기 (`assets/js/game/loop.js`)**
   - Step1에서 RUN 상태 고정 및 사이클 타이머 유지, Step1~2 동안 스톰 비활성화.
   - 튜토리얼 모드에서 이동/대쉬 감지, STOP 판정 성공/실패에 따라 안전 카운트 증가 및 즉시 재시도 처리.
   - `checkTutorialProgress()` 훅을 추가하여 성공 조건 충족 시 단계 전환을 트리거하고 runtime.tutorialStep을 동기화.

2. **사망 처리 우회 (`assets/js/game/lifecycle.js`)**
   - 튜토리얼 모드에서 `handleDeath` 호출 시 일반 사망 로직 대신 튜토리얼 재시도(`retryStep`)로 전환하도록 분기했습니다.

## 메모
- 일반 모드 흐름에는 영향을 주지 않도록 기존 로직을 보존하면서 튜토리얼 모드에서만 분기하도록 구현했습니다.
