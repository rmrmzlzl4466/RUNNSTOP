# 로비 캐릭터 표시 수정 및 확장

## 작업 단계
1. **스킨 데이터 검토**: 커스텀 스킨(Devil) 경로가 실제 파일 구조와 맞지 않아 로비 프리뷰가 비어 보일 수 있음을 확인했습니다.
2. **프리뷰 로직 보강**: 스프라이트 사용 시에도 기본 색상·네온 테두리를 유지하고, 이미지 로드 실패 시에도 컬러 원이 보이도록 로비 캐릭터 렌더링 로직을 정리했습니다.
3. **UI 크기 개선**: 로비 무대와 캐릭터 프리뷰, 스킨 이름 텍스트 크기를 키워 가시성을 높였습니다.

## 결과
- 스킨 스프라이트 경로가 올바른 파일을 가리키며, 로비 캐릭터가 안정적으로 표시됩니다.
- 스프라이트 로드 실패나 컬러 스킨 모두에서 기본 색상/테두리가 유지되어 빈칸으로 보이지 않습니다.
- 로비 캐릭터 프리뷰와 스킨 이름이 더 크게 표시되어 시인성이 향상되었습니다.
