# Tutorial Step 14 - Navigation sound unlock and delayed tutorial entry

## 작업 과정
- 오디오 컨텍스트 초기화/재개 시 예외가 상위 로직을 끊지 않도록 `Sound` 모듈 전체를 점검하고 재작성했습니다.
- 화면 전환(SFX)과 자동 튜토리얼 진입 지점을 따라가며, 사용자 제스처 이후에만 실행되도록 흐름을 재구성했습니다.
- 튜토리얼 진입 시점의 `tutorial-lock` 적용 시나리오를 재정리해, 전환 실패 시에도 입력이 막히지 않도록 확인했습니다.

## 결과
- 오디오 재생이 브라우저 제약에 막혀도 게임 흐름이 중단되지 않고, 경고 로그만 남도록 처리했습니다.
- 화면 전환 시 SFX가 잠금 해제 이후에만 시도되고, 실패해도 전환 흐름을 방해하지 않습니다.
- 자동 튜토리얼은 첫 사용자 제스처 이후에만 시작되며, 전환 성공 시점에만 `tutorial-lock`을 적용합니다.

## 테스트
- 별도의 자동 테스트는 수행하지 않았습니다 (수동 플로우 검증).
