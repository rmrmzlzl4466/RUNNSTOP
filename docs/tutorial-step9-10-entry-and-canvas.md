# Tutorial Expansion – Step 9 & 10

## 작업 요약
- 튜토리얼 자동/수동 진입 트리거를 `main.js`에 연결해 최초 실행 시 자동 진입, 로비 버튼으로 재진입이 가능하도록 했습니다.
- 튜토리얼 모드 전용 캔버스/런타임으로 전환하고, 완료 시 로비로 복귀하며 일반 게임용 런타임을 재구성하도록 처리했습니다.
- 튜토리얼 완료 시 런타임/네비게이션/저장 플로우를 안전하게 종료하도록 `TutorialManager`를 보강했습니다.

## 세부 변경 내역
1. **튜토리얼 진입 트리거 (`assets/js/main.js`)**
   - 저장 데이터에서 `tutorialCompleted`를 읽어 최초 실행 시 자동 진입 플래그(`window.shouldStartTutorial`)를 설정.
   - `TutorialManager`/`TutorialUI` 초기화 및 로비 `#btn-tutorial` 클릭 시 튜토리얼 시작, 타이틀 진입 시 자동 진입 분기 추가.
   - 튜토리얼 모드에서는 전용 캔버스(`game-canvas-tutorial`)로 런타임/라이프사이클을 재구성하고, 완료 콜백에서 일반 모드 런타임으로 복귀.

2. **튜토리얼 완료 플로우 (`assets/js/game/tutorial.js`)**
   - 완료 시 `runtime.tutorialMode` 해제, 단계/게임 활성화 상태 초기화, 슬로우모션 강제 종료.
   - 로비 화면으로 안전하게 이동하고, 외부 복구 콜백(`onTutorialFlowComplete`)을 호출해 일반 모드로 돌아갈 수 있게 함.

## 메모
- 튜토리얼 진입 시 게임을 자동으로 시작하지 않으며, 화면이 존재하지 않을 경우 콘솔 경고만 남기고 흐름을 중단하도록 방어했습니다.
- 튜토리얼 완료 후 일반 게임을 시작하려면 로비에서 기존 Start 버튼을 사용하면 됩니다.
