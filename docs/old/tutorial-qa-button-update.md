# QA Button Relocation and Styling

## 작업 단계
1. **브랜치 생성**: 기존 작업 브랜치에서 `tutorial` 브랜치로 분기했습니다.
2. **레이아웃 수정**: 로비 화면 상단에 있던 QA 버튼을 제거하고, 푸터 버튼 스택 하단에 QA 버튼을 추가했습니다.
3. **스타일 정렬**: QA 버튼이 SHOP 버튼과 동일한 형태를 사용하도록 버튼 스타일 클래스를 공유하도록 CSS를 수정했습니다.

## 결과
- QA 버튼이 로비 푸터에서 SHOP 버튼 바로 아래에 위치하며 동일한 시각적 스타일을 가집니다.
- 상단 코너의 QA 버튼 레이아웃 및 관련 스타일 정의를 제거했습니다.
