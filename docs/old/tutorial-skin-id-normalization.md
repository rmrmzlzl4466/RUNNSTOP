# 로비 스킨 표시 오류 수정

## 작업 단계
1. **세이브 데이터 정규화**: 기존 세이브에 문자열로 저장된 스킨 ID를 모두 숫자로 변환하고, 장착 스킨이 해제되지 않도록 기본 스킨을 강제로 추가했습니다.
2. **게임/코어 저장소 동기화**: 게임 런타임용 저장 모듈과 코어 저장 모듈 모두에 동일한 정규화 로직을 반영해 로비·게임 로딩 흐름을 일관되게 했습니다.
3. **로비 렌더링 보정**: 로비 캐릭터 렌더링에서 스킨 ID를 숫자로 변환해 찾고, 매칭 실패 시 기본 스킨을 사용하도록 안전장치를 추가했습니다.

## 결과
- Red/Blue Dot 등 다른 스킨을 장착하면 로비 캐릭터 UI가 즉시 해당 색상/스프라이트로 표시됩니다.
- 이전 버전에서 저장된 문자열 기반 세이브도 자동으로 수복되어 회색 기본값으로 고정되는 문제가 해소됩니다.
